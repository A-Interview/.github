## 🚀 How to Start
```
$ git clone --recursive [https://github.com/A-Interview/Docker.git]
```
### env setting in the Settings folder 
* settings/.env
```
# backend env
GPT_API_KEY=

DJANGO_SECRET_KEY=
MYSQL_DATABASE=
MYSQL_NAME=
MYSQL_USER=
MYSQL_PASSWORD=
MYSQL_ROOT_PASSWORD=
MYSQL_PORT=
MYSQL_HOST=

# AWS env
MY_AWS_ACCESS_KEY =
MY_AWS_SECRET_ACCESS_KEY =
AWS_STORAGE_BUCKET_NAME = 
AWS_S3_CUSTOM_DOMAIN =
FILE_URL=

```
### Run Docker
```
$docker compose up --build
```
